Table to Relation
=================

![Table to Relation widget icon](icons/table-to-relation.png)

Converts data table into a relation matrix. Labels matrix axes.

Signals
-------

**Inputs**:

- **Data**

  Attribute-valued data set.

**Outputs**:

- **Relation**

  Relations between two groups of objects.
